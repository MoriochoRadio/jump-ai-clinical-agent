# Reviewer Dashboard

This folder contains static, read-only reviewer views for committed scenario outputs.

Current dashboard:

- `scenario_002_review.html`

Open the HTML file directly in a browser. It does not require a server, backend, package install, login, network call, patient data, or EMR/HIS access.

The dashboard is not the source of truth. The committed CLI outputs under `prototype/runs/` remain the source artifacts.
