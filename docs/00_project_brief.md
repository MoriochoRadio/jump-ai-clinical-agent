# Project Brief

## Working Title

JUMP AI Clinical Agent

## Purpose

Use the 4th JUMP AI / AI drug development challenge as a structured portfolio project connected to Medical IT, hospital information systems, healthcare data, and clinical workflow automation.

## Initial Hypothesis

The best fit is likely not a molecule-generation-only project. A stronger portfolio direction is an agentic AI system that helps clinical or hospital-facing drug development work, such as:

- clinical trial protocol planning,
- regulatory/guideline compliance review,
- hospital data collection readiness,
- evidence-backed risk reporting.

## Why This Is Worth Doing

The project can show more than model training:

- understanding of healthcare operations,
- ability to design reliable AI workflows,
- documentation discipline,
- public data/API integration,
- software engineering around traceability and evaluation.

## Success Criteria

Minimum successful portfolio outcome:

- a clear GitHub repository,
- competition analysis,
- selected problem definition,
- agent architecture,
- evaluation plan,
- prototype demo or mock workflow,
- proposal draft.

Competition success outcome:

- strong preliminary proposal,
- credible evaluation design,
- possible advancement to finals,
- final demo with service link/report/slides if selected.
