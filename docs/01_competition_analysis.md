# Competition Analysis

## Competition

제 4회 인공지능(AI) 신약개발 경진대회, 4th JUMP AI / fourth.py 예선.

Official challenge page:

- https://daker.ai/public/hackathons/4th-jump-ai-agentic-drug-challenge

Official public notice:

- https://www.khidi.or.kr/board/view?linkId=48946784&menuId=MENU01108

## Core Question

> 당신의 에이전틱 AI는 신약개발의 어떤 문제를 해결하는 전문가입니까?

## What Is Submitted In The Preliminary Round

The preliminary round is proposal-centered.

Expected submission:

- proposal file in the provided hwpx form,
- within 10 pages excluding attachments,
- includes selected field, team name, agent name, keywords, proposal content, and related items.

This is not a typical leaderboard-only competition where the main action is training a model and submitting a prediction CSV.

## Preliminary Evaluation

Preliminary scoring combines:

- peer review: 30%,
- expert review: 70%.

Important expert criteria:

- necessity and problem definition,
- originality and creativity of agent design,
- technical feasibility,
- appropriateness of evaluation set and metrics,
- business and social value,
- research ethics and completeness.

## Final Round

If selected for the final round, teams build an actual AI agent and submit:

- demo video,
- research/technical report,
- service deployment link,
- presentation materials.

Final evaluation focuses on:

- scientific validity and innovation,
- agent autonomy and intelligence,
- external tool/API/database integration,
- resource efficiency,
- demo completeness and transparency.

## Initial Interpretation

The competition rewards a credible agentic service concept more than raw model-training effort. A strong proposal should prove that:

1. the problem matters in real drug development or clinical research,
2. agentic AI is genuinely useful for the problem,
3. the system can use tools and external evidence,
4. outputs can be evaluated and audited,
5. the implementation is realistic.
