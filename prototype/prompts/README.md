# Prototype Prompts

This folder will hold prompt templates for the first workflow.

The MVP uses separate prompt files so that each agent role can be reviewed and improved independently.

Planned prompt files:

- `protocol_checklist.md`
- `trial_case_evidence.md`
- `hospital_data_readiness.md`
- `critic_safety.md`
- `final_report.md`

The first implementation should keep prompts conservative, source-aware, and explicit about safety boundaries.
