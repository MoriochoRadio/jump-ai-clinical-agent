# Research Notes

Use this folder for source-backed notes.

Possible research areas:

- clinical trial protocol design,
- ICH/FDA/MFDS guidance,
- ClinicalTrials.gov data model,
- PubMed/PubChem/ChEMBL public APIs,
- hospital data readiness and EMR/HIS workflow,
- AI agent evaluation.

Each research note should include:

- source URL,
- access date,
- short summary,
- relevance to proposal,
- cautions or limitations.
