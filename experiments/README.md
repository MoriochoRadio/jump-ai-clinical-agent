# Experiments

Use this folder for small experiments.

Potential experiments:

- query ClinicalTrials.gov for a disease area,
- retrieve PubMed abstracts,
- create a protocol checklist,
- test a simple agent routing flow,
- evaluate outputs with a fixed rubric.

Each experiment should document:

- goal,
- input,
- method,
- result,
- limitation,
- next action.
